VoiceFlow TTS

Hybrid TTS Tool: Web Speech API + Google TTS

URL API 參數:

- text: 文字內容

- auto=1: 使用 Google TTS 自動播放 (無介面)

- download=1: 使用 Google TTS 直接下載 MP3

- lang: 語言 (預設 zh-TW, 支援 en, ja...)

- rate: 語速 (0.1 ~ 10.0, 映射至 ttsspeed)

註: API 模式不支援音調與選擇聲音，且建議少於 200 字。
